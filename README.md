# Healthy Recipes

Healthy Recipes is an android application designed to help users find and save healthy and delicious meals.

## Features
- Browse a curated list of healthy recipes
- Search recipes by name or description
- View detailed recipe information
- Mark recipes as favourites
- Persistent local storage for favourites
- Light and dark theme support

## Architecture
- UI and functionality was built with Jetpack Compose in Android Studio
- ViewModel manages UI state
- Repository handles data persistence
- Navigation handled using Compose Navigation

## Technologies Used
- Kotlin
- Android Studio
- Jetpack Compose
- Android ViewModel
- Local file persistence
- Gradle version catalog

## How to run
- Compile the project using gradle and run through an emulator or physical device.
